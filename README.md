# Name of project
social-dapp-stellar
# Features
On-chain Profiles: Users can set up a unique name and bio tied directly to their wallet address.
Decentralized Posting: Create permanent, timestamped posts.
Identity Verification: Uses require_auth to ensure only the owner of a key can post or update their profile.
